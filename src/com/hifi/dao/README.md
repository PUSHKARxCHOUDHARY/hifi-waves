# dao Package

DAO (Data Access Object) classes manage all interactions with the MySQL database.

---

## Files:

###  UserDAO.java  
Contains methods for:
- Registering a new user
- Validating login credentials

###  MessageDAO.java  
Handles:
- Saving chat messages to the database
- Retrieving all messages for display

Using DAO ensures a clean separation between business logic and database operations.
