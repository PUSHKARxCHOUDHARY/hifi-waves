# util Package

This folder contains general-purpose helper utilities used throughout the application.

### Files Included:

#### DBConnection.java  
Provides a reusable method for connecting to the MySQL database using JDBC.  
Centralizing connection logic prevents code duplication.

#### PasswordUtil.java  
Handles password hashing using SHA-256 to enhance login security and avoid storing plain-text passwords.

These utilities increase project quality and demonstrate understanding of reusable components.
