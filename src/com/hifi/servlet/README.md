# servlet Package

This folder contains all Servlets responsible for managing user interaction and request flow.

---

## Servlets Included:

###  RegisterServlet  
Displays registration page and handles new user creation.

###  LoginServlet  
Authenticates user credentials and manages session creation.

###  ChatServlet  
Loads chat page, retrieves messages, and saves new messages.

###  LogoutServlet  
Ends the user session properly and redirects to login.

Each servlet acts as a controller and interacts with DAO + JSP pages.
