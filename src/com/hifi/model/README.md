# model Package

This folder includes POJO (Plain Old Java Object) classes that represent data structures used in the system.

---

## Files:

###  User.java  
Represents an application user with properties like id, username, and email.

### Message.java  
Represents a chat message with message id, user id, username, content, and timestamp.

These classes make data transfer between DAO and Servlets clean and object-oriented.
