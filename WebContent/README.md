# WebContent Folder

This is the frontend layer of the application, containing JSP pages, CSS, and web configuration files.

---

##  Folders

### jsp/  
Contains all JSP files used as the user interface:
- login.jsp  
- register.jsp  
- chat.jsp  

### css/  
Optional folder for stylesheets.

###  WEB-INF/  
Contains:
- web.xml (deployment descriptor)
- lib folder (MySQL JDBC Driver JAR)

This folder is essential for application security and deployment.
